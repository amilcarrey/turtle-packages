# @turtledev/api

The official TypeScript SDK for interacting with the Turtle API.

## Overview

This package provides a type-safe client for all Turtle API endpoints, including:

- **Organization management**
- **Deal operations**
- **User authentication**
- **TVL data queries**

## Configuration

The API client uses a global configuration that can be set once:

```typescript
import { config } from "@turtledev/api";

config.set({
  apiKey: process.env.TURTLE_API_KEY,
  baseUrl: "https://api.turtle.club",
});
```

## Request Signing

All requests are automatically signed using the provided API key for security.

## Endpoints

### Organization

- `organization.get()` - Get organization details
- `organization.deals()` - Get organization deals

### Deals

- `organizationsDeals.get()` - Get all organization deals
- `organizationDeals.get()` - Get specific organization deals

### Authentication

- `prepareSignup()` - Prepare signup process
- `signup()` - Complete signup
- `exists()` - Check if user exists

### TVL

- `projectTvl.get()` - Get project TVL data

## Type Safety

All endpoints are fully typed with TypeScript and validated with Zod schemas.
